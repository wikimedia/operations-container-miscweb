Comment 6 Dscho 2005-01-03 22:13:26 UTC

A MediaWiki extension which supports on-the-fly creation of musical notation

Hi,

I created a MediaWiki extension which uses LilyPond (in safe mode) to produce
musical notation. For short pieces, i.e. to describe a musical concept, you can
use

<lilypond>c4 e g</lilypond>

If you want to hear it by clicking on the resulting picture, just use
<lilymidi> instead of <lilypond>.

And the best: you can also use it to write whole pieces in a team, by using
<lilybook> instead of <lilypond>. The contents are then interpreted as a
LilyPond file, and the pages are output as pictures.

I did my own MediaWiki extension rather than use wikitex, because LilyPond is
steering clear of TeX (text and music do have different dimensions).

Ciao,
Dscho

Another smaller issue is: if ABC is used, should it be extended so that it is more user-friendly? Community input is needed on this.
