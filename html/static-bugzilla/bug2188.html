Comment 1 Andreas Kasenides 2005-05-17 06:35:07 UTC 
Actually what is meant here is that the follοwing correction should be applied
to NS_TEMPLATE and NS_TEMPLATE_TALK in LanguageEl.php.
From 
NS_TEMPLATE         => 'Φόρμα',
NS_TEMPLATE_TALK    => 'Συζήτηση_φόρμας',

should become
NS_TEMPLATE         => 'Πρότυπο',
NS_TEMPLATE_TALK    => 'Συζήτηση_προτύπου',


Also $wgSiteName should be changed from 
$wgSiteName = 'Wikipedia';
to
$wgSiteName = 'Βικιπαίδεια';

Everything else should fall in place then.
