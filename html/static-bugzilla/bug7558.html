Wikimedia has migrated from Bugzilla to Phabricator. Bug reports should be created and updated in Wikimedia Phabricator instead. Please create an account in Phabricator and add your Bugzilla email address to it.

You could still run searches in Bugzilla or access your list of votes but bug reports will obviously not be up-to-date in Bugzilla.

Description Kaveh 2006-10-13 08:59:37 UTC 
Please create an Author and Author_talk namespace, a Page and Page_talk
namespace, and a Portal and Portal_talk for the Persian Wikisource
(fa.wikisource.org). As follows:

* Author -> مؤلف
* Author_talk -> بحث_مؤلف

* Page -> برگه
* Page_talk -> بحث_برگه

* Portal -> درگاه
* Portal_talk -> بحث_درگاه

Also for fa.Wikisource, please switch the logo with the current version (Wiki.png).
