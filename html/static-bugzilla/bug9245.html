<p><b>Technorati</b> is an <a href="/wiki/Internet"
title="Internet">Internet</a> <a href="/wiki/Search_engine" title="Search
engine">search engine</a> for searching <a href="/wiki/Blog"
title="Blog">blogs</a>, competing with <a href="/wiki/Google"
title="Google">Google</a>, <a href="/wiki/Yahoo" title="Yahoo">Yahoo</a> and <a
href="/wiki/IceRocket" title="IceRocket">IceRocket</a>. As of December 2006,
Technorati <a href="/wiki/Web_indexing" title="Web indexing">indexes</a> over 55
million weblogs. The name Technorati is a <a href="/wiki/Portmanteau"
title="Portmanteau">portmanteau</a>, pointing to the <a href="/wiki/Technology"
title="Technology">technological</a> version of <a href="/wiki/Literati"
title="Literati">literati</a> or intellectuals.</p>
<p>Technorati was founded by <a href="/wiki/Dave_Sifry" title="Dave Sifry">Dave
Sifry</a> and its 
59b
<a href="/wiki/Headquarters" title="Headquarters">headquarters</a> are in <a
href="/wiki/San_Francisco%2C_California" title="San Francisco, California">San
Francisco</a>, <a href="/wiki/California" title="California">California</a>, <a
href="/wiki/United_States" title="United States">USA</a>. <a
href="/wiki/Tantek_%C3%87elik" title="Tantek Ã‡elik">Tantek Ã‡elik</a> is the
site's Chief Technologist.</p>
<p>Technorati uses and contributes to <a href="/wiki/Open_source" title="Open
source">open source</a> software. Technorati has an active <a
href="/wiki/Software_developer" title="Software developer">software
developer</a> community, many of them from open-source culture. Sifry is a major
open-source advocate, and was a founder of <a href="/wiki/LinuxCare"
title="LinuxCare">LinuxCare</a> and later of <a href="/wiki/Wi-Fi"
title="Wi-Fi">Wi-Fi</a> access point software developer Sputnik. Technorati
includes a public developer's <a href="/wiki/Wiki" title="Wiki">wiki</a>, where
developers and contributors collaborate, as well as various open <a
href="/wiki/API" title="API">APIs</a>.</p>
<p>The site won the <a href="/wiki/SXSW" title="SXSW">SXSW</a> 2006 awards for
Best Technical Achievement and also Best of Show. It has also been nominated for
a 2006 <a href="/wiki/Webby" title="Webby">Webby</a> award for Best Practices.</p>
