Description Purodha Blissenbach 2007-05-13 21:16:38 UTC 
There is an experimental keyboard translator for the Amharic and several Indic 
scripts. It adds a checkbox and a hint text in front of the page edit box in 
several Wikipedias, e.g.:
"Phonetic Bangla Typing. Check/Uncheck box or use Esc as a toggle key to 
activate/deactivate. Go to WP:Typing for help" in the Bangla Wikipedia.

A space should be inserted at the end of this message and before the edit box 
in the html of edit pages. Else, the last word is concatenated with the left 
edge of the edit box, thus the exit box moves right. If a user has selected to 
have the edit box use the full width, moving the edit box to the right ( in a 
clipping box surounding it ) makes some Browsers clip the scroll bar of the 
edit box. This is highly undesirable.

The missing space results in a uggly or broken page layout, anyways, no matter 
what the user preferences are.
